Gives a video effect to the pictures. ImageMation widget lets you put a set of sliding images with captions on any of your wordpress page/post. 

== Description ==

Gives a video effect to the pictures. Show a set of sliding images which interacts with your mouse movement on your wordpress pages/posts .
You can specify the window size  and the big image will slide with in that window rectangle. Add up to 10 images of any dimensions for animation.  
Captions can also be specified for each image . The image can be moved to any four directions with the movement of the mouse. if the mouse is moved in north-east direction , 
the image also starts moving in the same direction. Animation pauses with Mouseover and starts again with Mouseout.

**Features:**

* Display big images in a small window
* Gives video effect to an image
* Image interact with the movement of mouse
* Specify custom images

**Usage:**

* Go to Appearance menu and click on widgets and add WP_ImageMation widget to your desired place.  


== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip WP-ImageMation.zip and Upload the complete directory `WP-ImageMation` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go To Wordpress Settings and Click on Wp_ImageMation
4. Specify Width, Height and URL's & Captions of the images.
5. Go to Appearance menu and click on widgets and add WP_ImageMation widget to your desired place.  
4. This is it ! A window of the size you specified will appear at that place with moving images in it giving it a video effect.

Alternatively if you don't want to add it as an widget then you can place the following code :- 

 `<div id="slidingimage"'></div>
 <div id="infopane"'></div>`
 
 on any page or post you wish to display the sliding images.

5. Optionally you can change the caption color & positioning by editing <b>#infopane</b> values located in <b>ImageMation.css</b> file in the plugin directory.
